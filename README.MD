# 🏠 Favelas - Autenticación con OTP

Sistema de inicio de sesión que envía códigos de un solo uso (OTP) al correo electrónico del usuario.  
Construido con **FastAPI** (backend) y **JavaScript/HTML/CSS** (frontend).

---

## ✨ Funcionalidades

- ✅ Envío de código OTP por correo electrónico
- ✅ Verificación del código ingresado
- ✅ Interfaz amigable y responsive
- ✅ Conexión entre frontend y backend
